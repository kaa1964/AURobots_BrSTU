# Регламент соревнований «Проплыви по коридору»
**Данное описание регламента не является официальным документом. 
Назначение текста - обозначить направления чемпионата.
Окончательный регламент после принятия оргкомитетом будет представлен 
участникам чемпионата )**

## 1. Общие положения

### 1.1. Цели и задачи
- Разработка алгоритмов навигации в ограниченном пространстве
- Отработка движения по заданной траектории

### 1.2. Термины и определения

| Термин  | Определение                                              |
|---------|----------------------------------------------------------|
| Коридор | Пространство между двумя жёлтыми линиями на дне бассейна |
| Старт   | Зелёный круг диаметром 0.5м                              |
| Финиш   | Красный круг диаметром 0.5м                              |

## 2. Технические требования

### 2.1. Поле соревнований
- Бассейн размером: **ширина 6м, длина 10м** [citation:7]
- Глубина воды: 0.35м от дна до поверхности
- Материал дна: тёмно-серый (контрастный)

### 2.2. Разметка
| Элемент	    | Цвет   | Расположение |
|-------------------|--------|--------------|
| Левая линия	    | Жёлтый | X = 0        |
| Центральная линия | Жёлтый | X = 1        |
| Правая линия      | Жёлтый | X = 2        |
| Старт             | Зелёный| X = 1, Z = -4|
| Финиш             | Красный| X = 1, Z = 4 |

### 2.3. Требования к роботу
- Автономное движение
- Обнаружение линий (датчики цвета/ИК) [citation:7]

## 3. Порядок проведения

### 3.1. Формат соревнований
- Каждая команда выполняет **3 попытки**
- В зачёт идёт лучшее время

### 3.2. Штрафы
| Нарушение               | Штраф     |
|-------------------------|-----------|
| Выход за линию коридора | +5 секунд |
| Пропуск старта          | Перезапуск|

## 4. Система оценки

| Критерий            | Баллы                   |
|---------------------|-------------------------|
| Прохождение трассы  | 100                     |
| Время (секунды)     | бонус: (эталон - время) |
| Соблюдение коридора | 50                      |
